====================
3. Say my name module:
====================

Import module:
>>> say_my_name = __import__('3-say_my_name').say_my_name

Function:
=======.
Write a function that prints My name is <first name> <last name>

Operations:
=========================
>>> say_my_name("Lina", "Montano")
My name is Lina Montano

>>> say_my_name("", "Holberton")
My name is  Holberton

>>> say_my_name("Gladiz", 'Ramirez Ardila')
My name is Gladiz Ramirez Ardila

=============================

>>> say_my_name("Vanessa", "")
My name is Vanessa 

>>> say_my_name("", "")
My name is  

>>> say_my_name("Dolar")
My name is Dolar
